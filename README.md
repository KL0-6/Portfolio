# My portfolio website!

This is the source code behind my [website](https://kl0-6.com/)! 

## Stack

The stack I used for this website is:
- React.js
- Typescript
- Vite
 
## Build and Test

- Firstly install all required NPM packages.
```bash
npm i
```
- To test
```bash
npm run dev
```
- To build
```bash
npm run build
```
